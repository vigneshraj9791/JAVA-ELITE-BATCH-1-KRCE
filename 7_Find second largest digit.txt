11_Swap first and last digit

Swap the first and last digits by extracting them using division and modulo, keeping the middle part unchanged, and recombining them in reversed positions.

Program:
import java.util.*;
class Main{
    public static void main(String[] args){
        Scanner sc=new Scanner(System.in);
        int a=sc.nextInt();
        int temp=a;
        int pow=1;
        int last=a%10;
        while(temp>=10){
            temp/=10;
            pow*=10;
        }
        int first=temp;
        int middle=(a%pow)/10;
        int result=(last*pow)+(middle*10)+first;
        System.out.println(result);
    }
}
