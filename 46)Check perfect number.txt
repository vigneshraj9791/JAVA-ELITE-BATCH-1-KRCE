45)Find sum of factors:

The sum of factors of a number is the sum of all integers that divide the number exactly (including 1 and the number itself).

To compute this efficiently, iterate from 1 to √n.

If i divides n, then both i and n/i are factors.

Add both to the sum.

If i == n/i, add it only once to avoid duplication.

This reduces time complexity to O(√n)

PROGRAM:

class Solution {
    public int sumOfFactors(int n) {
        int sum = 0;
        for (int i = 1; i * i <= n; i++) {
            if (n % i == 0) {
                sum += i;
                if (i != n / i) {
                    sum += n / i;
                }
            }
        }
        return sum;
    }
}
