50)Find largest prime factor

Given a number n, your task is to find the largest prime factor of n.

PROGRAM:

class Solution {
    public long largestPrimeFactor(long n) {
        long maxPrime = -1;
        
        while (n % 2 == 0) {
            maxPrime = 2;
            n /= 2;
        }
        
        for (long i = 3; i * i <= n; i += 2) {
            while (n % i == 0) {
                maxPrime = i;
                n /= i;
            }
        }
        
        if (n > 2) {
            maxPrime = n;
        }
        
        return maxPrime;
    }
}
