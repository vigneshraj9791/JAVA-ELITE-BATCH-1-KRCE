45.Find sum of factors

Given an integer n, your task is to find the sum of all its factors (divisors).

A factor of a number is any number that divides it exactly (without remainder).

import java.util.*;

public class Main {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int sum = 0;

        for (int i = 1; i <= n; i++) {
            if (n % i == 0) {
                sum = sum + i;
            }
        }

        System.out.println(sum);
    }
}
